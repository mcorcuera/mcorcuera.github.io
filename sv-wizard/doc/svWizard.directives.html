(static) confirmAction :Directive

The confirmAction (confirm-action) displays a confirmation dialog one the user click into the element it is inserted in. It only executes the action if the user click in the Accept or Yes button.

The action to execute upon confirmation is provided in the attribute confirm-action and the message to display to the user in the confirm-action-message.

(static) mapAutocomplete :Directive

The mapAutocomplete (map-autocomplete) directive has to be applied in an input method. It will provide the Google Maps autocomplete feature to this input element. Once the user selects a place, the function provided in the directive value will be called with the selected result.

(static) mapPreview :Directive

The mapPreview (map-preview) directive inserts a Google Map into the view linked to a Street View panorama so it updates its position when the panorama does and also, it changes the panorama location when the pegman is dropped some where.

The following attributes are included in the scope
  • map: contains the Google Map object created within the directive
  • panorama: the Street View panorama object to update the position

(static) svPreview :Directive

The svPreview (sv-preview) directive element creates a Street View Panorama object in the view, and links the positions and view direction parameters to the ones provided to the scope. That means that changes in the Street View will be reflected in the provided values and changes in the values will be reflected into the Street View Panorama.

It exposes the following parameters as attributes
  • panorama: contains the Google Panorama object created within the directive
  • location: the LatLng coordinates of the Street View
  • heading: the heading value of the Street View
  • pitch: the pitch value of the Street View
  • fov: the fov value of the Street View
  • size: the size of the request. It will be used to calculate the aspect ratio of the request and use the same one to display the Street View.
