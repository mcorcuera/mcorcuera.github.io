Class: Settings

svWizard.services.Settings

new Settings(localStorageService)

The Settings service is used to store and retrieve the configuration of the application using the browser's local storage. Its properties are automatically saved from local storage upon change.
It is included in the angular app module as Settings service and needs no configuration
